# Jiayi Mao Portfolio Site

Static personal portfolio website for Jiayi Mao, shaped around a dark analytics-terminal visual language inspired by the information architecture of Xiaoxian Wang's portfolio. The content is based on Jiayi's complete resume and highlights marketing analytics, business intelligence, data quality, reporting, and decision support.

## Pages

- `index.html` - home page with positioning, quantified evidence, featured projects, operating model, and experience preview
- `about.html` - education, target roles, and working approach
- `projects.html` - four analytics case studies
- `experience.html` - professional experience reframed by business problem, data, tools, and outcome
- `skills.html` - skill categories by workflow
- `resume.html` - online resume summary and PDF download
- `contact.html` - public contact details

## Assets

- `assets/img/analytics-dashboard.png` - homepage dashboard visual
- `assets/img/jiayi-mao.jpg` - profile photo
- `assets/resume/Jiayi-Mao-Resume-Complete.pdf` - downloadable resume from the supplied PDF

Open `index.html` directly in a browser, or run a simple static server from this folder:

```bash
python3 -m http.server 5173
```
